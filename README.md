# ALHE project

1. Create a virtual environment

Make sure to use python 3.5

```
virtualenv --python=/usr/bin/python3.5 venv
source venv/bin/activate
```

To deactivate

```
deactivate
```

2. Run the program

```
python src/main.py
```

3. Run the tests

```
python -m unittest discover -s=src
```
