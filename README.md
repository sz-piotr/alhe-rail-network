# ALHE project

1. Create a virtual environment

Make sure to use python 3.5

```
virtualenv --python=/usr/bin/python3.5 venv
source venv/bin/activate
```

To deactivate

```
deactivate
```
2. Install the dependencies

```
pip install -r requirements.txt
```

3. Run the program

```
./run
```

4. Run the tests

```
./test
```
